# Paso 0 (Si no lo tiene)
1.- Descargar, instalar GIT
2.- Descargar, instalar VSC
3.- Crear un directorio para trabajar y abrirlo con VSC.
4.- Abrir la terminal y ejecutar:
    git config --global user.name "nombre_usuario" ==> agregar nombre de usuario
    git config --global user.email "correo" ==> agregar el correo del usuario

# Paso 1
1.- Dentro de un directorio clonar el proyecto
    git clone http:\\.....git
2.- Abren la carpeta clonada con VSC
3.- Para guardar los cambios
    git Add . ==> prepara los archivos (stagging)
    git commit -m "cometario de lo que realizado" ==> crea un punto de guardado
    git push origin main ==> sube los archivos a GH
